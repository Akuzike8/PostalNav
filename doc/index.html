Search

URL: /search/{slug}

Method: GET

Description: Retrieves postal information for a given slug.

Parameters:

  • slug (string): The unique identifier for the postal information.
GET /search/{slug} 
                    
                        {"Id":"fbfae471-044c-4c15-82ff-7f9c06a1f5b9","Area":"Chemusa","Postal_code":"312209","District":"BLANTYRE","Region":"Southern Region","Settlement":"Urban"}
